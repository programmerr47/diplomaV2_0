Все операции с таблицами и вычисления представлены в вкладке «Вычисления». С помощью вложенной в нее вкладки «Метод вычисления» можно выбрать метод, который будет применяться при интерполировании и экстраполировании данных. С помощью кнопки «Вычислить» запускается процесс вычисления данных. С помощью кнопки «Сохранить вычисления» можно скопировать откорректированные данные из результирующей таблицы в исходную таблицу для проведения дальнейших, более точных вычислений. 

В программе на данный момент используются два метода вычисления: метод Лагранжа и кусочно-линейный метод.
