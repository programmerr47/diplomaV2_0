Вся работа с файлами представлена в виде кнопок «Новый проект», «Загрузить данные», «Сохранить данные», «Сохранить как…», которые находятся во вкладке «Файл». Их значение интуитивно понятно из названия. 

Таблицы в программе(их две; одна для известных точек, A; другая для точек с неизвестной индукцией, B) сохраняется в пользовательские файлы, используя формат представления CSV, который легко можно открыть, например, с помощью Microsoft Excel. Данные хранятся в следующем формате (поэтому сам файл можно легко заполнить с помощью обычного текстового редактора): 

<Данные>;<Данные>;<Данные>;<Данные>;<Данные>;<Данные>
<Данные>;<Данные>;<Данные>;<Данные>;<Данные>;<Данные>
<Данные>;<Данные>;<Данные>;<Данные>;<Данные>;<Данные>
<Данные>;<Данные>;<Данные>;<Данные>;<Данные>;<Данные>
NEXT
<Данные>;<Данные>;<Данные>;<Данные>;<Данные>;<Данные>
<Данные>;<Данные>;<Данные>;<Данные>;<Данные>;<Данные>

Где данные до NEXT – это записи первой таблицы; данные после NEXT – это записи второй таблицы.

Обычно программа запоминает, с каким файлом она работала в прошлый раз, поэтому при запуске она пытается открыть запомненный ей файл. Если этого не получается, она, уведомляя пользователя соответствующим сообщением, начинает работать с новым документом. 

Следить за тем, с каким файлом работает программа, можно в верхней части окна программы, где записано имя текущего открытого файла.

При выходе программа автоматически запрашивает разрешение сохранить текущие данные. Если программа работает с определенным файлом, данные автоматически сохранятся туда, если же программа работает с новым документом, то сначала придется выбрать директорию, а так же имя файла, куда будут сохранены все данные.
